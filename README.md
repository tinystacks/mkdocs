# TinyStacks Documentation

## Installation
```
pip install mkdocs
pip install mkdocs-material
```

## Making changes

1. git checkout gh-pages
2. git pull
3. git checkout master
4. --- make changes ----
5. mkdocs build, mkdocs serve (to run locally)
6. --- commit changes ---
7. git push
8. mkdocs gh-deploy --clean
